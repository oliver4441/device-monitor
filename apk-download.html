Installation Steps

  1. Download the APK file using the button above
  2. Open the downloaded file on your Android device
  3. Allow "Install from unknown sources" if prompted
  4. Grant all requested permissions (location, storage, etc.)
  5. Open the app — it will auto-register with the server
  6. The device appears in the dashboard automatically

Requirements: Android 8.0 (API 26) or higher. Battery optimization should be disabled for reliable background monitoring.
